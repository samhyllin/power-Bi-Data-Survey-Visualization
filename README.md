# Power-Bi-Data-Survey-Visualization

## Background

This project analyzes a data professional survey dataset using Power BI. The objective of the analysis is to understand trends among data professionals across different countries, job roles, and demographic groups.

The dataset contains information about survey respondents including job titles, countries, favorite programming languages, age distribution by sex, and salary levels.

The goal of the dashboard is to transform raw survey data into clear visual insights that help explain workforce trends, compensation patterns, and technology preferences within the data industry.
